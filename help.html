Complete documentation for the AlaeAutomates API.

What This API Does

This API provides three main services:

  • Statement Processing: Processes PDF bank statements and matches company names against a Do Not Mail (DNM) list from an Excel file.
  • Invoice Processing: Extracts invoice numbers from PDF files and splits them into separate documents.
  • Credit Card Batch Processing: Processes Excel files with credit card data and generates enhanced JavaScript automation code for Legacy Edge browsers.
API Endpoints

Base URL

Production: https://alaeautomatesapi.up.railway.app

Local: http://localhost:8000

Health Check

GET /health

Statement Processing

  • POST /api/statement-processor - Create Session
  • POST /api/statement-processor/{session_id}/upload - Upload Files
  • POST /api/statement-processor/{session_id}/process - Process Files
  • GET /api/statement-processor/{session_id}/questions - Get Questions
  • POST /api/statement-processor/{session_id}/answers - Submit Answers
  • GET /api/statement-processor/{session_id}/download - Download Results
  • GET /api/statement-processor/{session_id}/status - Session Status

Invoice Processing

  • POST /api/invoice-processor - Upload and Process
  • GET /api/invoice-processor/downloads/{zip_filename} - Download ZIP

Credit Card Batch Processing

  • POST /api/credit-card-batch - Upload and Process
